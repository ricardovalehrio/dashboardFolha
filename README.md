# 📊 Dashboard Interativo de Folha de Pagamento

Este projeto é uma solução de Business Intelligence desenvolvida em **Python** para análise dinâmica de folhas de pagamento. Ele transforma dados brutos de planilhas Excel em indicadores visuais e insights gerenciais, facilitando a tomada de decisão.

## 📋 Funcionalidades
- **Indicadores Chave (KPIs):** Visualização instantânea de média salarial líquida, total de horas extras e descontos.
- **Filtros Dinâmicos:** Filtragem por nível funcional e por nome do colaborador na barra lateral.
- **Gráficos Comparativos:** Análise de Salário Bruto vs. Líquido e impacto acumulado de descontos.
- **Relatório Detalhado:** Tabela interativa com ordenação automática e formatação de moedas.

## 🛠️ Tecnologias Utilizadas
- **Linguagem:** Python 3.x
- **Análise de Dados:** Pandas
- **Interface Web:** Streamlit
- **Manipulação de Excel:** OpenPyXL
- **Versionamento:** Git & GitHub
- **Hospedagem:** Streamlit Cloud

## 📂 Estrutura do Projeto
- `app.py`: Código principal da aplicação Streamlit.
- `extraFinal.xlsx`: Base de dados (Excel) utilizada para alimentar o dashboard.
- `requirements.txt`: Lista de dependências para execução do projeto.
- `.gitignore`: Arquivo para evitar o envio de arquivos temporários ao repositório.
