# 📊 Dashboard de Folha de Pagamento

Este projeto é um dashboard interativo construído com **Python** e **Streamlit** para análise de dados de folha de pagamento a partir de uma base Excel.

## 🚀 Como executar
1. Instale as dependências: `pip install -r requirements.txt`
2. Execute o app: `streamlit run app.py`

## 🛠 Tecnologias
- Python, Pandas, Streamlit.
